Flow 3 - Byg en app

Project description

In this flow we had to construct a prototype for a mobile app in Adobe Xd. This project was in groups.

We had to choose a company or organization and imagine that we had to construct an app for them. An important element of this app: We had to incooperate one or more of UN's Sustainable Development Goals. A big focus in this project was teamwork and groupwork.
